# 📘 Spring Boot + Thymeleaf Practice Projects

This repository contains multiple **Spring Boot + Thymeleaf** practice examples created to understand server-side rendering and MVC architecture.

Each folder demonstrates a specific Thymeleaf concept integrated with Spring Boot.

---

## 📂 Project Structure

Thymeleaf_Example-1  
Thymeleaf_Example-2  
Thymeleaf_Example-3  
Thymeleaf_Example-4  
...  
Thymeleaf_Example-16  

Each example is an independent Spring Boot project focusing on a particular Thymeleaf feature.

---

## 🚀 Technologies Used

- Java (JDK 21 LTS)
- Spring Boot
- Thymeleaf
- Spring MVC
- Maven
- HTML5
- CSS

---

## 🎯 Learning Objectives

Through these examples, I practiced:

- Configuring Thymeleaf in Spring Boot
- Passing data from Controller to View
- Using Model attributes
- Expression Language (`${}`)
- `th:text`, `th:if`, `th:each`
- Form handling with Thymeleaf
- Path variables
- Conditional rendering
- Template fragments
- Dynamic URL mapping

---

## 🧠 Key Concepts Covered

- MVC (Model-View-Controller) Architecture  
- Loose Coupling in Spring  
- Dependency Injection  
- Server-side Template Rendering  
- Request Mapping and View Resolution
